# Authentic Capstone Guide

An authentic Capstone project is a real software engineering effort where students work through ambiguity, make engineering decisions, collaborate with stakeholders, and deliver a meaningful system over one semester.

Use this guide to quickly answer three questions before proposing or reviewing a project.
