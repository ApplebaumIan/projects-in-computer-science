# Javadoc export tutorial

Short walkthrough showing how to export Javadocs from IntelliJ and host them in this Docusaurus site.

Use the pages below for a step-by-step guide and common gotchas:

- [Step-by-step tutorial](./tutorial.mdx)
- [Troubleshooting & tips](./troubleshooting.mdx)

:::tip New Inline Docs Component
You can now use the `<InlineDocs>` component to embed the Javadoc directly in your MDX pages. This is a great way to provide quick reference without needing to open a new tab. Check out the [component documentation for usage details](https://applebaumian.github.io/tu-cis-4398-docs-template/tutorial/custom-components/inline-docs).
:::
