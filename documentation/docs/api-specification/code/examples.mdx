# Examples and Previous Class Specs

This page contains example class specs and real-world examples you can copy and adapt.

## Example: UserService

Class: UserService
Purpose: Business logic for user account management.

Fields:
- userRepository: UserRepository — storage layer (private)
- logger: Logger — used for audit logs (private)

Methods:
- createUser(req: CreateUserRequest): User — creates a user
  Preconditions: req.email is a valid email, password meets complexity
  Postconditions: user persisted, welcome email enqueued
  Throws: ValidationException on invalid input
  Example:
  ```java
  UserService svc = new UserService(userRepository, logger);
  User u = svc.createUser(new CreateUserRequest("a@b.com", "P@ssw0rd"));
  ```

- getUser(id: UUID): User — returns the user or throws NotFoundException
